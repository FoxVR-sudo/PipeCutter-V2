# PipeCutter-V2

Version 2 of the PipeCutter project - A CNC pipe cutting automation system.

## Overview

PipeCutter-V2 is designed to facilitate automated pipe cutting operations using CNC (Computer Numerical Control) technology. This project aims to provide software and tools for precise pipe cutting automation.

## Features

- CNC pipe cutting automation
- Support for complex pipe geometries
- Integration with CNC control systems
- Automated GCODE generation for pipe cutting operations
